This is a CRM application that supports various endpoints that can be accessed with the following requests:

  • A GET request through /customers that retrieves all customers
  • A POST request through /customers that creates a new customer
  • A PATCH request through /customers/{id} that updates an existing customer
  • A DELETE request through /customers/{id} that deletes an existing customer
  • A GET request through /customers/{id} that retrieves an existing customer
